많이들 다짐하지만 오답노트 작성은 만만치 않죠.

🤔 문제랑 출처를 옮겨적기가 시간이 오래걸려요.

🤔 오답노트에서 원하는 복습문제를 찾기 번거로워요.

🤔 다시 풀기 위해 책을 또 구매해요.

위 고민들 중 2가지 이상 해당된다면,

      문제를 직접 옮겨적던 단순노동은 이제 그만

✅ 다양한 폰트와 테마로 꾸미기
